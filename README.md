# TaleOfFactualEval
Dataset and Codebase for ACL 2025 Findings

For the dataset and system predictions, please unzip the file **“A_Tale_of_Evaluating_Fact_dataset.zip.”**

The archive contains two subdirectories:

- **Dataset**: Includes the original documents, summaries, and system predictions.
- **System_Predictions**: Contains the system prediction labels and the computed metrics for different systems on each dataset.
